Demultiplexeur:

Types:

Demux 1 vers 2, Demux 1 vers 4, Demux 1 vers 8, Demux 1 vers 16 et Demux 32 vers 1.

Exemple: Demux 1->4.

En entree:   1 bit.

En commande:  2 bits.

En sortie:  4 bits.

Equation0:   sortie (S0) = C0\.C1\.E.

Equation1:   sortie (S1) = C0\.C1.E.

Equation2:   sortie (S2) = C0.C1\.E.

Equation3:   sortie (S3) = C0.C1.E.

Table de verité (4 sorties):

C1 C0 S0 S1 S2 S3
0 0 E 0 0 0
0 1 0 E 0 0
1 0 0 0 E 0
1 1 0 0 0 E

Note :

En cas de liaison de la sortie avec une des entrees il faut qu'au moins une de ses autres entrées soit a 0 .
