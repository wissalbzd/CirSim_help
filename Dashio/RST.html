La bascule RST

En entree :  3 bits

    R : Première entrée de la bascule sur 1 bit.

    S: Deuxième entrée de la bascule sur 1 bit.

    T : Entrée de commande(souvent c'est l'horloge où la bascule devient synchrone). 

En sortie :  2 bits

    Q : Sortie de la bascule sur 1 bit(résultat de l'évaluation).  

   !Q : l'inverse de la sortie Q  

Table de transition

Entrees Sorties
T R S Q !Q

Synchrone

 M 0 0 précedent !précedent
M 0 1 1 0
M 1 0 0 1
M 1 1 0 0

Asynchrone

 0 X X précédent !précedent
1 0 0 précedent !précedent
1 0 1 1 0
1 1 0 0 1
1 1 1 0 1

M :L'état de l'horloge est au mode de la bascule.

X :Quelle que soit la valeur (1 ou 0),elle n'effectue pas le résultat.

Notes   :
   La valeur initiale de la sortie Q avant la première évaluation du circuit est a 0.
   Les entrées Pr et Clr sont:
   Asynchrones.
   PLus prioritaires que le mode synchrone(Quand Jk != 11 la bascule est asynchrone).
   Fonctionnent en logique négative.
